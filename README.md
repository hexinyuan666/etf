ETF每日评级系统
一个基于多因子模型的ETF量化评级系统，为本人参加ETF菁英挑战赛设计。

🎯 项目简介
通过动量、波动率、风险调整收益和趋势质量等四大因子，对全市场400+只ETF进行综合评分和排名，为投资决策提供数据支持。

✨ 核心功能
多因子评分：动量(35%) + 波动率(20%) + 风险调整(25%) + 趋势质量(20%)

完整覆盖：分析全市场400+只ETF

智能推荐：自动生成前3名推荐标的

挂单建议：基于ATR波动率提供买卖价格区间

调仓建议：持仓管理和风险控制建议

🛠 快速开始
安装依赖

bash

pip install -r requirements.txt

运行系统

bash

python etf_daily_rating_complete.py


📊 输出内容
综合排名：前50名ETF详细评分

类别排名：按宽基/行业/跨境/商品分类展示

推荐持仓：前3名优质ETF

挂单建议：基于波动率的买卖价格区间

调仓策略：仓位管理和风险控制建议

⚙️ 核心因子
因子类别	权重	包含指标
动量因子	35%	1月/3月/6月动量、趋势斜率
波动率因子	20%	年化波动率
风险调整收益	25%	夏普比率
趋势质量	20%	ADX指标、200日均线
📁 文件说明
etf_daily_rating_complete.py - 主程序文件

etf_holdings.json - 持仓记录文件

etf_complete_rating_YYYYMMDD_HHMM.csv - 完整排名结果

etf_top100_rating_YYYYMMDD_HHMM.csv - 前100名结果

⚠️ 风险提示
本系统仅为量化分析工具，不构成投资建议。市场有风险，投资需谨慎。
